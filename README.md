# Smart File Organizer

An intelligent, automated agent to organize your file system safely.

## Features
- **Safe by Default**: Runs in "Dry Run" mode unless explicitly told to execute.
- **Clean Architecture**: Decoupled logic ensures stability.
- **Deduplication**: Cryptographic hash-based duplicate detection.
- **Smart Sorting**: Sort by Date or Extension.
